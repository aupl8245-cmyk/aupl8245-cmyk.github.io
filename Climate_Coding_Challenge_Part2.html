# Keep only DATE and TMAX, rename for clarity
df = df[['DATE','TMAX']].rename(columns={'TMAX':'temp_c'})

# Convert to numeric and scale if needed
df['temp_c'] = pd.to_numeric(df['temp_c'], errors='coerce')

# NOAA TMAX is often in tenths of °C — scale down
if df['temp_c'].max() > 200:   # crude check
    df['temp_c'] = df['temp_c'] / 10.0

# Drop missing values
climate_clean = df.dropna().copy()

# Add 5-year rolling mean
yearly['MA5'] = yearly['temp_c'].rolling(5, min_periods=1).mean()

plt.figure(figsize=(10,5))
plt.plot(yearly['Year'], yearly['temp_c'], label="Yearly Avg")
plt.plot(yearly['Year'], yearly['MA5'], label="5-Year Moving Avg", linewidth=2)
plt.xlabel("Year")
plt.ylabel("Max Temp (°C)")
plt.title("Boulder Yearly Max Temperature vs 5-Year Moving Avg")
plt.legend()
plt.show()

For Part 2, I cleaned the daily Boulder TMAX dataset, aggregated it into yearly averages, and calculated a 5-year moving average to smooth out short-term fluctuations. The moving average makes the long-term trend easier to see compared to raw yearly values.
